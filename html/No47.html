Presentation Topic

深度學習也可以學傅立葉轉換？

傅立葉神經算子

Can Deep Learning also learn Fourier Transform? Fourier Neural Operator

主講人 (Presenter): Yueh-Hua Tu

日期 (Date): 2020.12.4

核心概念：神經算子

Core Concept: Neural Operator

傳統神經網絡學習的是有限維度空間之間的映射。神經算子 (Neural Operator) 旨在學習無限維度函數空間之間的映射。

這對於解決偏微分方程 (PDEs) 等問題至關重要，因為 PDEs 的解通常是函數。

PDE 問題設定

Problem Setting for PDEs

目標是學習一個算子 L，將輸入函數/參數 a(x, t) 映射到解函數 u(x, t)。

(La u)(x) = f(x) , x ∈ D
u(x) = 0 , x ∈ ∂D (Boundary Condition)

其中 D 是有界開集 (bounded, open set)，La 是依賴於參數 a 的微分算子，f 是給定函數。

傅立葉轉換與格林函數

Fourier Transform & Green's Function

傳統上，傅立葉轉換可用於求解某些 PDE。例如，對於熱傳導方程：

ut = α2 uxx

透過傅立葉轉換，可以將偏微分方程轉換為常微分方程 (ODE)：

dû/dt = -ω2 α2 û(ω, t)

格林函數 (Green's function) Ga(x, y) 提供了另一種求解 PDE 的思路，解 u(x) 可以表示為格林函數與源項 f(y) 的積分：

u(x) = ∫D Ga(x, y) f(y) dy

註：û 表示 u 的傅立葉轉換，ω 是頻率。

相關研究方法

Related Work & Approaches

第一種方法 (First Approach)

  • 使用 CNN 作為算子 F: RK × Θ → RK
  • 依賴於網格 (Mesh-dependent)
  • 受限於訓練數據的離散化大小和幾何形狀
  • 無法查詢新的點 (Not able to query new points)

第二種方法 (Second Approach)

  • 將解 u 參數化為神經網絡 F: D × Θ → R
  • 不依賴於網格 (Mesh-independent)
  • 無法適應新的參數 (Not able to adapt to new parameters)

本研究方法

This Approach (Fourier Neural Operator)

  • 學習無限維度空間之間的映射
  • 對於函數的解析度不變 (Invariant to the resolution of the function)
  • 能夠轉移解 (Able to transfer solutions)

核心是將積分算子中的 Kernel 用神經網絡來參數化。

vt+1(x) = σ(W vt(x) + ∫D κφ(x, y, a(x), a(y)) vt(y) dy)

核函數與神經算子

Kernel Functions & Neural Operators

核函數

Kernel Function

或稱為積分核 (Integral Kernels)，其一般形式為：

(Tf)(u) = ∫ f(t)K(t, u)dt
  • 若算子 L 的核 (kernel of L) 非平凡 (non-trivial)，則格林函數 (Green's function) 不唯一。
  • 透過對稱性或邊界條件的組合，可以得到唯一的格林函數。
  • 當格林函數唯一時，它本身就是一個核函數。

卷積核

Convolution Kernel

格林函數可為卷積核：

G(x, s) = G(x - s)

這意味著算子具有平移不變性 (translation invariant)，類似於卷積神經網路 (CNN) 的特性。卷積核是核函數的一個重要範例。

神經算子概覽

Neural Operator Overview

神經算子學習從一個函數空間到另一個函數空間的映射。其核心流程可表示為：

a(x) → P → v₀(x) → [W, ∫κ] → ... → vT(x) → Q → u(x)

其中 a(x) 是輸入函數，u(x) 是輸出函數，中間經過提升 (P)、迭代更新 (W, ∫κ) 和投影 (Q) 等步驟。

提升層 (P)

Lifting Layer

將輸入資料（座標 x、函數 a(x) 及其平滑版本 a_ε(x) 與梯度 ∇a_ε(x)）投影到高維潛在空間：

v₀(x) = P[x, a(x), aε(x), ∇aε(x)]T + p

維度映射：ℝ2(d+1) → ℝn

P 是一個 n × 2(d+1) 的矩陣。

核積分迭代

Kernel Integration Layer

在（概念上的）無限維空間中，透過核積分進行迭代更新：

vt+1(x) = σ(Wvt(x) + ∫B(x,r) κφ(x, y, a(x), a(y))vt(y)dy)
  • W: 潛在空間中的線性變換。
  • κ_φ: 可學習的核函數，參數化為 φ。
  • 積分在以 x 為中心、半徑為 r 的球體 B(x,r) 上進行。

投影層 (Q)

Projection Layer

將高維潛在表示投影回目標解空間：

u(x) = QvT(x) + q

維度映射：ℝn → ℝ1 (若輸出為純量)

Q 是一個 1 × n 的矩陣。

圖核網路 (GKN)

Graph Kernel Network

GKN 使用消息傳遞神經網路架構來近似核積分部分：

vt+1(x) = σ(Wvt(x) + (1/|N(x)|) Σy∈N(x) κφ(e(x,y))vt(y))
  • 消息函數 (Message): M(x, y, e) = κφ(e(x,y))vt(y)
  • 邊特徵 (Edge Feature): e(x,y) = (x, y, a(x), a(y))
  • 聚合函數 (Aggregate): 平均 (Mean)
  • 更新函數 (Update): U(x, m) = σ(Wvt(x) + m)

圖構建

Graph Construction for GKN

基於積分域 B(x,r)，將中心點 x 與其半徑 r 內的鄰近點連接成圖結構。

Nyström 近似

Nyström Approximation

用於將連續積分運算元離散化。對於 RKHS 中的核 κ：

Tf = ∫ κ(·, y)f(y)ν(dy)

可近似為：

Tf ≈ (1/m) Σk=1m κ(·, yk)f(yk)

其中 y_k 是 m 個採樣點，δ_(y_k) 是狄拉克函數。

傅立葉神經算子 (FNO)

Fourier Neural Operator

FNO 的核心思想是利用傅立葉變換實現全局卷積。其核積分部分近似為：

∫ κφ(...)vt(y)dy ≈ F-1(Rφ ⋅ F(vt))(x)
v(x) FFT (F) × Rφ (頻域濾波) IFFT (F-1) + Wv(x) σ (激活)
  • F 表示傅立葉變換，F⁻¹ 表示逆傅立葉變換。
  • R_φ 是在傅立葉空間中可學習的線性變換（通常是對角矩陣），用於加權或篩選不同頻率模式 k。
  • 此操作等效於在空間域中的卷積。

FNO 討論

FNO Discussion

  • 激活函數位置 (Activation Function): 通常在空間域 (spatial domain) 中應用，即在逆傅立葉變換之後。
  • 邊界條件 (Boundary Condition): FNO 不嚴格要求週期性邊界條件，這使其適用性更廣。
  • 循環結構 (Recurrent Structure): FNO 層可以堆疊，形成類似循環神經網路的結構，用於處理時間演化問題。
  • 應用領域 (Applications): 不僅限於偏微分方程 (PDEs)，也可應用於計算機視覺等領域，例如將圖像視為函數進行處理。
